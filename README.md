# MDNA TECH TEST

### Prerequisites

php 7


## Running the tests

```
./vendor/bin/phpunit --bootstrap vendor/autoload.php --testdox tests
```

### Installing
Run Composer Update

## Running the scripts

Roman to Arabic (parse)

```
php index.php -p XCVII
```


Arabic to Roman (generate)

```
php index.php -g 1974
```
